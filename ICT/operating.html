OPERATING SYSTEMS

An operating system is an organised collection of software which controls the management of a computer system’s resources. It is the main collection of systems software that enables the computer system to manage the resources under its control. This group of programs manages the computer’s activities. It is the most critical piece of software in the computer system, which plays a central role in coordinating all of the computer’s work, and without an operating system, no other program can run.

The primary chores of an operating system are management and control. The operating system ensures that all actions requested by a user are valid and processed in an orderly manner. It also manages the computer system’s resources to perform those operations with efficiency and consistency. The operating system carries out three main functions, namely internal management, data management and job control to achieve management and control roles.

Bootstrapping

A bootstrap operation loads the operating system’s programs into the computer’s internal storage. This is accomplished when the computer operator inputs the address of the disc cartridge on which the operating system is stored, bringing into storage a small segment (the nucleus) of the total operating system. This portion assumes control automatically “bootstrapping” (or pulling in) the balance of the operating system known as leaves into the computer’s internal memory.

Two of the Operating System’s Features

Storage protection

This feature prevents one program from accessing internal storage areas occupied by other programs.

Priority

This feature establishes the relative importance of each concurrent program

Three main Functions of an Operating System


i. Getting programs ready for execution.
ii. Controlling programs after they start executing.
iii. Moving data into and out of internal storage. Control language supply data to the operating system.

Computer Management in a Multiprogramming Environment

An Operating System performs three management functions:

i.

Job Control (or management of jobs)

The various individual tasks that the operating system performs under this heading include:


(a) Analysing the input.
(b) Preparing a job for execution.
(c) Obtaining system resources.
(d) Analysing and processing operator commands.
(e) Transmitting messages from the operating program to the Operator.
(f) Processing job terminations.

ii.

Internal Management (management of internal operations or task management)

These functions involve supervising the interface between hardware and software:


(a) Processing Interrupts.
(b) Scheduling and supervising program execution.
(c) Allocating space in internal storage.
(d) Managing dynamic storage.
(e) Supervising clocks and timers.

iii.

Data Management (or management of data)

This function involves the interfacing between external storage and processing programs:


(a) Assigning storage on direct access devices.
(b) Maintaining a catalogue of files.
(c) Providing support routines for I/O operations.
(d) Handling I/O operations
